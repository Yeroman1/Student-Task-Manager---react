## Project Overview

The Student Task Manager is a sleek, responsive dashboard designed to help students organize their daily academic and personal tasks. It features a clean, card-based user interface with smooth transitions, dark mode support, and persistent local storage. The visual progress bar keeps users motivated by tracking their daily completion rate.

## Features
- **Add/Edit/Delete Tasks**: Full CRUD functionality for task management.
- **Dark Mode**: Built-in toggle for comfortable viewing in any lighting condition.
- **Local Storage**: Automatically saves your tasks and theme preferences to the browser.
- **Progress Tracking**: A dynamic progress bar that calculates your daily completion rate.
- **Responsive Design**: Carefully crafted to look perfect on mobile, tablet, and desktop screens.
- **Filtering & Search**: Easily find tasks by text search or by their active/completed status.
- **Due Dates & Overdue Highlights**: Keep track of deadlines with visually distinct overdue indicators.

## Technologies Used
- **React 18** (Functional Components & Hooks)
- **Vite** (Build Tool & Dev Server)
- **Tailwind CSS v3** (Utility-first Styling)
- **Lucide React** (Modern SVG Icons)
- **Date-fns** (Date manipulation and formatting)
- **Clsx & Tailwind-merge** (Dynamic class merging)



## Installation Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/student-task-manager.git
   cd student-task-manager
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   Navigate to `http://localhost:5173` to view the application.
